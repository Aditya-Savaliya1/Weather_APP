Weather App

This Weather App provides real-time weather information for any city worldwide. It uses a public weather API to fetch live data, including temperature, humidity, wind speed, and weather conditions. The interface is simple, fast, and easy to use.

Features

Search weather details by city name

Real-time temperature, humidity, and wind speed

Weather condition icons and clear text descriptions

Simple and clean user interface

Fast and accurate weather updates

Technologies Used

Python

Requests library (for API calls)

JSON data handling

OpenWeatherMap API (or your preferred weather API)

Tkinter / HTML-CSS-JS (depending on your version)
